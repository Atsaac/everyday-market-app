# Everyday Market App

## Description
This is an Angular standalone component application built for the Everyday Market assignment.
It contains:
Angular routing
Standalone components
Reactive forms
Validation
Dependency injection issues
Layout styling

## Features
- Header component visible on all pages
- Category menu with selectable items
- Parent-child communication using Input and Output
- Clean folder structure and reusable components

## How to Run
1. Install Node.js
2. Install Angular CLI: npm install -g @angular/cli
3. Navigate to the project folder
4. Run: ng serve
5. Open browser at http://localhost:4200
